What happened?

What happened to PhoenixCraft?

Phoenixcraft, was once a Minecraft network created with a dream to take off and create memorable experiences for all of its players.

After thorough decision, I've decided that due to a lack of playerbase, life getting busy, and our current lovely worldly situation, that I'm going to shut down PhoenixCraft permanently.

PhoenixCraft may return in the future, but next time I'm going to ensure that I have a community backing before I go about doing this again.

Server Statistics

Server (Host) Information:

PhoenixCraft was hosted locally in Saint Louis, MO on a Dell R710 via FTTP (Fiber to the premises) service under a hypersecure environment.

The server was operated in a VM allocated the following specifications:
  • Dual x5670 Xeon Processors (4 cores allocated)
  • 24GB of server memory
  • 256GB+ (Expandable upon necessity)
  • Fibre optic internet access with speeds reaching 1000/1000 (Upload/Download)
The server reached the following statisics throughout it's lifetime:
  • 16 unique players on survival, and 6 unique players on skyblock.
  • We actually managed to beat the game. (Slayed the end dragon)
  • We successfully deployed a multi-server setup. (I Personally never expected to do this.)
